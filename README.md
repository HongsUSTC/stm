Plotting measurements using Gnuplot
===================================

Prepare measurements in a file `measurements`, with one measurement per line:

    0.0
    0.0
    0.0
    0.1
    0.2
    0.3

Plot:

    plot 'measurements' using 1


Coding conventions
==================

* Maximum line length: 80 characters

* Comments in Markdown
