Instructions
============

 1. Upload code to Teensy 3 or compatible.

 2. Start server:

        $ cd server
        $ node app.js

 3. Open `app.html` in browser.


Coding conventions
==================

* Maximum line length: 80 characters

* Comments in Markdown

* JavaScript passes JSLint.
